## 📚 오픈소스 소프트웨어 강의 요약

### 1️⃣ **오픈소스 소프트웨어 개요**
- **정의**: 소스 코드를 모두가 접근할 수 있도록 공개한 소프트웨어.
- **특징**:
  - 협업과 지속적인 개선 장려.
  - 버전 관리를 위해 Git, 저장소 호스팅을 위해 GitHub 사용.

### 2️⃣ **Git과 GitHub**
#### **Git**
- 파일 변경 이력을 추적하는 분산 버전 관리 시스템.
- 주요 명령어:
  - `git init`: 저장소 초기화.
  - `git add`: 변경 사항 스테이징.
  - `git commit`: 변경 사항 커밋.
  - `git push`: 원격 저장소로 업로드.

#### **GitHub**
- Git 저장소 호스팅 및 협업을 위한 웹 기반 플랫폼.
- 주요 기능:
  - Pull Requests
  - 코드 리뷰
  - 이슈 및 프로젝트 관리

### 3️⃣ **주요 학습 내용**
- 버전 관리 시스템: 로컬 및 원격 저장소.
- Git 명령어: `add`, `commit`, `push`, `pull`, `log`, `diff`.
- GitHub을 활용한 협업 개발.
- 브랜치와 병합 기본 개념.
